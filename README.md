## 📌 Overview
The **Async Weather Tracker** is a simple web application built using **Vanilla JavaScript** that fetches real-time weather data using a public API.  
It demonstrates core concepts of **asynchronous programming**, **Fetch API**, **Promises**, and **Local Storage**.

---

## 🎯 Features

- 🔍 Search weather by city name  
- 🌡️ Displays temperature and weather condition  
- ⚡ Uses **async/await** for API calls  
- ❌ Handles errors (invalid city, network issues)  
- 💾 Stores search history using **Local Storage**  
- 🔁 Click previous searches to fetch again  
- 🧠 Console logs to understand **event loop & execution order**

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript (Vanilla)** – Logic & API handling  

---

## 🚀 How It Works

1. User enters a city name  
2. Clicks the **Search** button  
3. App fetches data from weather API  
4. Displays:
   - City name  
   - Temperature  
   - Weather condition  
5. Saves search in Local Storage
